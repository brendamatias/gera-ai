## 🧰 Instalação

```bash
# Clone o repositório
git clone https://github.com/brendamatias/gera-ai.git

# Acesse a pasta do projeto
cd gera-ai

# Instale as dependências
npm install

# Rode o projeto localmente
npm run dev
```
